# Pillyliu.com Brand Style

This document defines the long-term shared header style for all site sections.

## Header system
- Use the same header layout, sizing, typography, spacing, and button/chip styling across all pages.
- Header has two text lines on the left:
  - Line 1 (small uppercase label): section brand
  - Line 2 (larger bold title): page-specific title

## Section brand line (small uppercase)
- Default root brand: `PILLYLIU`
- Section variants:
  - Pinball section: `PILLYLIU PINBALL`
  - Pizza section: `PILLYLIU PIZZA`
  - Processors/gaming section: `PILLYLIU PROCESSORS`
- Keep this pattern for future domains/sections.

## Page title line (larger bold)
- Line 2 should describe the page itself.
- Examples:
  - `Peter Liu` (landing)
  - `League Stats`
  - `League Standings`
  - `League Targets`
  - `Pinball Library`

## Navigation
- Navigation button visual style should be consistent site-wide.
- Each section can have a custom nav list, while keeping identical button/chip formatting.

## Background treatment
- Keep a subtle top-focused gradient for visual identity.
- On long pages, let the lower page settle into mostly dark background.
