<div class="max-w-3xl mx-auto p-8 print:p-4"><h1 class="text-2xl font-bold text-dark-green mb-4">Evel Knievel SS - Bally | 1977</h1><div class="flex gap-6 mb-6"><div class="flex-shrink-0"><img src="https://img.opdb.org/a5119f54-3a55-4e03-b7f8-6cd3c83adc30-medium.jpg" alt="Evel Knievel backglass" class="w-80 h-auto"/></div><div class="flex-1"><p class="mb-1"><strong>Shortname:</strong> EK</p><p class="mb-1"><strong>Players:</strong> 4</p><p class="mb-1"><strong>Display:</strong> alphanumeric</p><div class="mt-3 p-3 bg-gray-100 rounded"><h2 class="text-lg font-bold text-dark-green mb-1">Quickie Version:</h2><p class="whitespace-pre-line capitalize-first-letter">shoot the left drops to light one of the spinners and double bonus, then lit spinner all day. If a spinner is lit to begin the game, just do spinner all day from the start.</p></div></div></div><section class="mb-4"><h2 class="text-lg font-bold text-dark-green mb-1">Go-to Flipper:</h2><p>Bias right early, then balanced</p></section><section class="mb-4"><h2 class="text-lg font-bold text-dark-green mb-1">Playfield Risk:</h2><p class="whitespace-pre-line">Bumper exits and rebounds from the drop targets on the left. Don’t shoot the stand-up targets, they’re not worth the rebound risk.</p></section><section class="mb-4"><h2 class="text-lg font-bold text-dark-green mb-1">Full Rules:</h2><div class="space-y-3"><p class="whitespace-pre-line capitalize-first-letter">This is one of those “deferred UTAD” games. First, you need to complete the set of five drop targets to light one of the spinners alternately. This scores 5,000 and lights Double Bonus, so you’ve got three excellent reason for doing this right away. Then you go UTAD through the lit spinner. Pay attention to the fact that balls rolling through a return lane will change the lit spinner to the side facing that flipper if that’s not already the lit side - - the machine sets up your shot for you\! Plan on shooting the spinner after any return lane balls - - cradle up if it’s going slowly enough, if not, one-time it.</p><p class="whitespace-pre-line capitalize-first-letter">Some Evel Knievels have one spinner lit from the start - - that’s the machine’s easier setting. If the one you’re playing is like that, just shoot the spinner all day from the start.</p><p class="whitespace-pre-line capitalize-first-letter">Base bonus maxes out at 15K, 30K with double bonus. A few good spinner shots can get you that much, so don’t sweat the bonus advances. The drop targets, stand-ups, top lanes and return lanes all advance bonus, so with a decent ball, you’ll max it out without trying to.</p><p class="whitespace-pre-line capitalize-first-letter">You get a letter in S-U-P-E-R for each shot to the top saucer; while it may be set to award 25,000 points for a special when completed, which letter it awards is random and the letters reset every ball, so it’s rare to finish them.</p><p class="whitespace-pre-line capitalize-first-letter">Completing the right side “C-L-E” targets lights the center bumper to be worth 1,000. Completing C-Y-C-L-E (the first C and the Y are the top side lanes near the saucer) lights one of the outlanes for a special, also worth 25,000. Like the SUPER letters, the CYCLE letters reset each ball, making them hard to safely finish. I tend to ignore them due to rebound risk from the C-L-E stand-ups; if you get them anyways, fine.</p></div></section><section class="mb-4"><h2 class="text-lg font-bold text-dark-green mb-1">Skill Shot:</h2><p class="whitespace-pre-line capitalize-first-letter">drop the ball in the top saucer for a letter in S-U-P-E-R.</p></section></div>
